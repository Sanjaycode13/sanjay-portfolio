Diabetes Mellitus Classification using KNN Algorithm

  • Tech Stack: Python, Scikit-learn, Pandas, NumPy, Matplotlib, Jupyter Notebook
  • Status: Completed

This project involves building a predictive model to classify whether a patient is diabetic based on various health-related metrics. Using a labeled dataset (such as the PIMA Indians Diabetes dataset), the system employs **K-Nearest Neighbors (KNN)** algorithm to determine a patient’s diabetic condition based on features like glucose levels, BMI, age, blood pressure, and insulin levels.

The project pipeline includes:

  • Data cleaning and preprocessing: Handling missing values and outliers
  • Feature scaling: Standardizing features to ensure accurate KNN distance calculations
  • Exploratory data analysis (EDA): Visualizing patterns and correlations using Matplotlib and Seaborn
  • Model training: Using scikit-learn’s KNN classifier with optimized k-values
  • Model evaluation: Calculating accuracy, confusion matrix, ROC-AUC score, and classification report

The model achieved promising accuracy in classifying diabetic vs non-diabetic patients and serves as an important proof-of-concept for deploying AI in healthcare diagnostics. Future enhancements include integration into a web app or mobile interface for real-time predictions.
