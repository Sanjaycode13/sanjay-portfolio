VOXA is a voice-controlled personal assistant built specifically to automate WhatsApp messaging without human intervention. Using speech recognition technology, it allows the user to send messages by simply speaking recipient names and message content. It integrates with Excel to match names with phone numbers and uses PyAutoGUI to control WhatsApp Desktop.

The system supports:

  • Real-time speech-to-text conversion
  • Excel contact matching and validation
  • Full automation via Python GUI scripting
  • Multi-contact broadcasting
This project successfully reduced manual input by over 90%, offering a scalable voice-messaging solution for large user groups and contact books.
